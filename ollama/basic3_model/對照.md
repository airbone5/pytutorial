
## v0 的執行結果
```
>>> 甚麼是保險
保險（保單）是為了保護個人或企業的財產和權益而設立的一種法定約合。它是一種合同，根據規定的條款，供給者將承擔部分風險，而購買者則以一定的付費額獲得這部分風險的抵消。

例如：假如您投資了一塊 Properties，而遭遇了火災，經過保單，您可以透過保單方的責任獲得這部分損失的補償。

保險除了可以保護個人財產外，也能夠減少企業的風險和資金流動。

甚麼是保險人
保險人（保單公司）是為了實現保險合同目的而設立的組織或機構。

保險人負責承擔保險合同所定的義務，包括支付損失或傷害。保險人的主要目的是為了盈利，通過收取保單費用、保險 Premium，而不是依靠投保人提供資金。

例如：您 purchase了一份生命 insurance，該公司就是您的保險人。當您逝世後，保險人將按照規定的條款支付一定的額度給您的 heirs。

在保險合同中，保險人和投保人都是共有意義的方角。
```
